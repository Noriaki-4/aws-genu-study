# 災害対策本部 初動対応マニュアル 2026
org: 共通
category: 防災
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 初動基準
市内で震度5強以上を観測した場合、災害対策本部を自動設置する。
震度5弱の場合は、危機管理監が被害状況を確認し、必要に応じて本部設置を判断する。

## 2. 参集
本部員は設置連絡を受けた後、原則60分以内に庁舎または指定拠点へ参集する。
交通途絶時は最寄りの公共施設に参集し、安否と対応可能業務を報告する。

## 3. 住民情報の取扱い
避難所運営で取得した住民情報は、避難者支援、安否確認、物資配布の目的に限って利用する。
外部提供は、本人同意がある場合または法令に基づく場合に限る。
