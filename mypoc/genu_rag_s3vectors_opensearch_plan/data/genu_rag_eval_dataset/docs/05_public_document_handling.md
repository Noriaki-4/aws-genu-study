# 公開文書・情報公開対応手順 2026
org: 共通
category: 文書管理
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 公開文書の分類
公開文書は、条例、規則、告示、会議資料、予算資料、入札公告、審議会議事録に分類する。

## 2. 非公開情報
個人識別情報、法人の競争上の利益を害する情報、審査・交渉・調査に関する未確定情報は非公開または黒塗り対象とする。

## 3. 公開期限
会議資料は会議開催日の2営業日前までに公開する。
審議会議事録は会議終了後30日以内に公開する。
入札公告は公告日から契約締結日まで公開を継続する。

## 4. 保存形式
PDFを正本とし、検索性確保のためHTMLまたはテキスト抽出データを併せて保存することが望ましい。
