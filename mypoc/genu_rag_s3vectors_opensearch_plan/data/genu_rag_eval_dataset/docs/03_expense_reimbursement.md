# 出張旅費・経費精算規程 2026
org: 共通
category: 経費
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 領収書提出
交通費、宿泊費、備品購入費の精算には領収書の添付を必要とする。
ただし、公共交通機関のICカード利用履歴で金額と経路が確認できる場合、領収書の添付を省略できる。

## 2. 提出期限
経費精算の提出期限は、支出日の翌月10日までとする。
年度末の3月分については、4月5日までに提出しなければならない。

## 3. 宿泊費上限
宿泊費の上限は、東京都23区内が1泊14,000円、政令指定都市が1泊12,000円、その他地域が1泊10,000円である。
災害派遣時は、財務課長の承認により上限を超過できる。
