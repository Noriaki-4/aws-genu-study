# 庁内AI・RAGサービス SLA運用案 2026
org: 情報システム部
category: AI基盤
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 利用時間
庁内AI・RAGサービスの標準利用時間は平日8時から23時までとする。
夜間23時から翌7時まではデータ取込、再インデックス、バックアップを優先する。

## 2. 応答時間目標
通常チャットの初回応答は10秒以内、RAG検索を含む回答は20秒以内を目標とする。
大容量PDFや表抽出を伴う回答は30秒以内を努力目標とする。

## 3. 障害通知
全庁影響の障害は検知後15分以内に一次報を発信する。
復旧見込みが30分を超える場合は、30分ごとに続報を発信する。

## 4. コスト管理
組織別の利用量、モデル別トークン数、Knowledge Base別検索回数を日次で集計する。
