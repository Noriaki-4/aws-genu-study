# 物品調達・契約事務マニュアル 2026
org: 共通
category: 調達
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 見積取得
予定価格が10万円未満の物品購入は、原則として1者見積でよい。
10万円以上50万円未満は2者以上の見積を取得する。
50万円以上130万円未満は3者以上の見積を取得する。
130万円以上は入札または公募型見積合わせを原則とする。

## 2. 随意契約
緊急の災害対応、既存システムとの互換性、著作権・特許権により供給者が限定される場合は、随意契約を認める。
ただし、理由書と価格妥当性の説明を契約書類に添付する。

## 3. 生成AI関連調達
生成AIサービスを調達する場合は、データ保存場所、学習利用の有無、監査ログ、閉域網接続可否を確認する。
