# 情報セキュリティ運用基準 2026
org: 共通
category: セキュリティ
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 端末持出
業務端末を庁舎外へ持ち出す場合は、端末管理台帳への記録と所属長承認を必要とする。
端末にはディスク暗号化、画面ロック、EDRを必ず適用する。

## 2. USB利用
USBメモリの利用は原則禁止とする。
例外的に利用する場合は、情報システム部門が貸与する暗号化USBのみを使用し、利用後はウイルスチェックを実施する。

## 3. 生成AI利用
生成AIへ入力してはならない情報は、個人情報、未公表の政策情報、契約前の調達情報、認証情報、秘密指定資料である。
公開済み資料、庁内FAQ、一般的な文案作成は利用可能である。
