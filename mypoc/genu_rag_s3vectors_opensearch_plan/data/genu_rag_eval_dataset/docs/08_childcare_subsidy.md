# 子育て支援補助金 申請要領 2026
org: 福祉部
category: 補助金
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 対象
市内在住で、0歳から5歳までの児童を養育する保護者を対象とする。
所得制限は設けない。

## 2. 補助額
保育用品購入費の補助上限は児童1人あたり年額18,000円である。
多胎児の場合は、児童ごとに上限額を適用する。

## 3. 申請期限
2026年度分の申請期限は2027年3月15日である。
領収書の日付が2026年4月1日から2027年3月10日までのものを対象とする。

## 4. 併給制限
国または県の同種補助を受けた費用は、本補助の対象外とする。
