# 全庁テレワーク実施要領 2026年度版
org: 共通
category: 勤務制度
effective_from: 2026-04-01
effective_from_num: 20260401

## 1. 対象者
テレワークの対象者は、常勤職員、再任用職員、会計年度任用職員のうち、所属長が業務遂行上支障がないと認めた者とする。

## 2. 実施上限
通常時のテレワーク実施上限は週2日までとする。
ただし、災害対応、感染症対策、庁舎改修その他特別な事情がある場合、所属長の承認により週4日まで拡大できる。

## 3. 申請期限
職員は、原則として実施予定日の3営業日前までにテレワーク申請を行う。
緊急時は事後申請を認めるが、翌営業日までに理由を記録する。

## 4. 禁止事項
住民情報、税情報、個人番号を含む資料を自宅端末へ保存してはならない。
紙資料を自宅へ持ち帰る場合は、所属長の個別承認を必要とする。
